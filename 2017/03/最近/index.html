最近的生活

发表于 2017-03-20

有一段时间没有写点东西了，就写写最近发生的事情吧!

前天的时候，沛沛被吓到了，然后还有点积食。
当了爸爸，就是一种不一样的人生体验啊!以前的时候，都是自己管好自己就好了，现在呢，有一个小小的家伙需要依靠你来养活，需要你来保护。当她什么都不会说，自己不舒服的时候只是会哭的时候，也一样的会手足无措。其实说实话，这个时候，媳妇的存在给了我最大的安心。因为我自己也是像一个没有长大的孩子一样，很多时候不能很好的照顾自己。家里的每个人都很在乎我，都很爱我，可是前一段时间我的表现却是不爱自己呀!
沛沛的事情说完了，说说自己现在的状态吧!
年前做好的学习和看书的计划，其实现在还都没开始啊。叹气。
兴致勃勃做好了一年的计划，可是没有实施的时间和动力。其实最基本的还是没有动力吧，时间就像海绵里的水，只要挤挤还是有的。每次晚上走回家以后趴在床上休息的时间呢？每次躺在床上看手机的时间呢？每次蹲厕所看手机浪费的时间呢？就是没有努力改变自己的动力吧。
想好好练字，总是不能坚持；想好好读书，可是翻开了看几页就再也不去碰；每次看见别人推荐说XX书好，总是第一时间买来却从来没有打开过。

# 生活 # 感想
怕了
写给沛沛
